   GAME ENGINE EDITOR - ПОРТАТИВНАЯ ВЕРСИЯ
═════════════════════════════════════════════════════════════════

✓ Работает на любом Windows
✓ Не требует инсталляции
✓ Можно копировать на USB флешку

ТРЕБОВАНИЯ:
───────────
- Python 3.8+ (скачайте с https://www.python.org/downloads/)
- PyQt5 (установится автоматически)

ЗАПУСК:
──────
1️⃣  Двойной клик на "start_engine.bat"
2️⃣  Или на "GameEngineEditor.vbs" (без окна консоли)

ПЕРВЫЙ ЗАПУСК:
──────────────
При первом запуске будут установлены зависимости (займёт 1-2 минуты).
После этого приложение запустится нормально.

ПРОБЛЕМЫ:
─────────
❌ "Python не найден"
  → Установите Python с https://www.python.org/downloads/
  → Обязательно выберите "Add Python to PATH"

❌ "ModuleNotFoundError: PyQt5"
  → Удалите файл cache и запустите снова
  → Или вручную: pip install PyQt5
