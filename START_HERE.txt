⚡ БЫСТРЫЙ СТАРТ (2 минуты):

ШАГОЛЬ 1️⃣  - Проверка Python:
────────────────────────────────────
Откройте командную строку/терминал и выполните:

  python --version

Должно показать: Python 3.8 или выше
Если Python не установлен, скачайте с https://www.python.org/downloads/

ШАГОЛЬ 2️⃣  - Установка зависимостей:
────────────────────────────────────
В этой же папке выполните:

  На Windows:
    pip install -r requirements.txt

  На Mac/Linux:
    pip3 install -r requirements.txt

ШАГОЛЬ 3️⃣  - Запуск приложения:
────────────────────────────────────
Выполните одну из команд:

  На Windows:
    python run_engine.py
    или просто двойной клик на start_engine.bat

  На Mac/Linux:
    python3 run_engine.py
    или ./start_engine.sh

Или используйте QUICKSTART для быстрой проверки:
    python QUICKSTART.py

🎯 ПЕРВЫЕ ДЕЙСТВИЯ:

1. ЗАПУСТИТЕ ПРИЛОЖЕНИЕ:
   python run_engine.py
   
   Откроется главное окно с редактором

2. СОЗДАЙТЕ ПЕРВЫЙ ОБЪЕКТ:
   Меню "Редактирование" → "Добавить спрайт (2D)"
   Объект появится на холсте в центре

3. ПЕРЕМЕСТИТЕ ОБЪЕКТ:
   Нажмите и перетащите объект левой кнопкой мыши

4. ОТРЕДАКТИРУЙТЕ СВОЙСТВА:
   Выберите объект → нижняя правая панель покажет его свойства
   Измените позицию, ротацию, масштаб

5. НАПИШИТЕ СКРИПТ:
   Откройте редактор скриптов (нижний центр)
   Нажмите "Новый скрипт" → выберите шаблон

6. СОХРАНИТЕ И ЭКСПОРТИРУЙТЕ:
   Ctrl+S для сохранения
   Меню "Экспорт" → выберите формат

⚙️ РЕШЕНИЕ ПРОБЛЕМ:

Проблема: "ModuleNotFoundError: PyQt5"
Решение: pip install PyQt5

Проблема: Приложение не запускается
Решение: 
  1. Проверьте Python: python --version
  2. Установите PyQt5: pip install PyQt5
  3. Запустите проверку: python check_installation.py

Проблема: Объекты не видны
Решение: Используйте колёсико мыши для масштабирования сцены

Проблема: Скрипт не сохраняется
Решение: Укажите имя файла в поле "Script name" перед сохранением
